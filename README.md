# Yanhe-Web 项目简介

本仓库用于存储 [RUC Student Portal](https://learn.ruc.edu.kg) 的前端代码。

RUC Student Portal 提供比延河课堂官方网站更简单直接直播课程服务。用户可观看全校任意直播课程，同时在上课时同步从屏幕录制中提取课程课件。
